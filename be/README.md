# Backend

Node.js와 Express 기반 백엔드 애플리케이션을 위한 디렉토리입니다.

## 예정 역할

- 다이어리 API
- 장소 API
- 카테고리 API
- 공유 방 및 초대 코드 API
- 데이터베이스 연동

## 설정 메모

- 패키지 매니저는 pnpm을 사용합니다.
- 데이터베이스는 PostgreSQL + PostGIS를 기준으로 준비합니다.
- 실제 Express 프로젝트 scaffold 시 `AGENTS.md`의 기술 스택과 작업 규칙을 먼저 확인합니다.

## 예정 명령어

```bash
pnpm install
pnpm dev
pnpm build
pnpm typecheck
pnpm test
```
