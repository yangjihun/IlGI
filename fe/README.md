# Frontend

Vue와 TypeScript 기반 프론트엔드 애플리케이션을 위한 디렉토리입니다.

## 예정 역할

- 메인 지도 화면
- 다이어리 작성 및 상세 화면
- 가보고 싶은 장소 목록 화면
- 공유 방 화면
- 지도 API 연동

## 설정 메모

- 패키지 매니저는 pnpm을 사용합니다.
- 실제 Vue 프로젝트 scaffold 시 `AGENTS.md`의 기술 스택과 작업 규칙을 먼저 확인합니다.

## 예정 명령어

```bash
pnpm install
pnpm dev
pnpm build
pnpm typecheck
pnpm test
```
