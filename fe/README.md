# Frontend

Vue와 TypeScript 기반 프론트엔드 애플리케이션입니다.

## 기술 스택

- Vue
- TypeScript
- Vite
- Vue Router
- pnpm

## 실행 명령어

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
pnpm test
```

## 구조

- `src/main.ts`: Vue 앱 생성 및 router 등록
- `src/router/`: 라우팅 설정
- `src/views/`: 화면 단위 컴포넌트
- `src/style.css`: 전역 스타일

## 현재 상태

현재는 초기 scaffold 단계이며, 메인 화면 기반과 기본 라우팅만 포함합니다.
